Karate

Karate - an open-source tool that combines API test-automation, mocks, performance-testing and UI automation into a single, integrated framework. It leverages BDD language-neutral syntax, which is easy for even non-programmers. Assertions and HTML reports are built-in and thus there isn't a need to import additional libraries. It allows to run tests in parallel.
